Open Agents — chosen direction (3c) + Quick & Pro

One unified shell over the agentic primitives — Agents, MCP Servers, Skills, and now Workflows — with a single Quick ↔ Pro toggle. Below: Quick mode (the exec/citizen — conversational, declarative) up top, then Pro mode (the developer — code, permissions, environments) underneath. Each mode is shown across three surfaces: Home, Skills, and Workflow building.
